# 자동화 테스트 가이드 2025

Python과 Selenium을 사용한 웹 자동화 테스트를 위한 초보자 친화적인 가이드 웹페이지입니다.

## 📋 사용 방법

1. **웹페이지 열기**
   - `index.html` 파일을 웹 브라우저로 열기만 하면 됩니다.
   - 더블클릭하거나 브라우저로 드래그 앤 드롭하세요.

2. **로컬 서버로 실행 (권장)**
   ```bash
   # Python 3가 설치되어 있다면
   python -m http.server 8000
   
   # 또는 Python 2
   python -m SimpleHTTPServer 8000
   ```
   그런 다음 브라우저에서 `http://localhost:8000`으로 접속하세요.

## 🎯 주요 기능

- **5개의 섹션**: 소개, 설치 및 설정, 기초 개념, 실전 예제, 팁 & 트러블슈팅
- **다크모드 지원**: 우측 상단 버튼으로 다크/라이트 모드 전환
- **코드 복사 기능**: 모든 예제 코드를 한 번의 클릭으로 복사
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
- **실전 예제**: 5개의 실제 사용 가능한 예제 코드

## 🚀 포함된 예제

1. **구글 검색 자동화** - 검색어 입력 및 결과 확인
2. **로그인 폼 자동화** - 폼 입력 및 제출
3. **여러 페이지 크롤링** - 여러 페이지에서 데이터 수집
4. **파일 다운로드 자동화** - 파일 다운로드 자동화
5. **스크린샷 찍기** - 웹페이지 스크린샷 저장
